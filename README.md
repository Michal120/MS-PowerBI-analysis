# MS PowerBI - analysis of company sales

This project focuses on a thorough analysis of the relational database of a fictional company that sells models of cars, boats, and airplanes.

Project objectives
**Company analysis:** identifying the most profitable employees
**Sales trend analysis:** identifying the best-selling product lines and model manufacturers.
**Customer allocation:** defining the country with the highest sales.
**Logistics and warehouse:** checking order status and delivery efficiency, describing inventory status.
**Financial overview:** identifying the employees with the highest revenue.

Technologies used
**SQL (MySQL):** Selecting data from the database and working with tables using JOIN, SELECT, CREATE VIEW, ORDER BY.
**Power BI:** Creating measures using DAX, cleaning and transforming data using Power Query, visualizing dashboards.

Key findings
The "Classic Cars" line has the largest share according to quantity in the warehouse.
The average shipping time for goods is 3 days.
Most models were sold in the USA.
